## Project Overview

This is a Spring Boot REST API application built using a layered architecture approach. 
The application exposes RESTful endpoints to perform CRUD operations and interacts 
with a relational database using Spring Data JPA.

## Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Architecture

Controller → Service → Repository → Database

## Features

- Create, Read, Update, Delete operations
- RESTful API design
- Exception handling
- Database integration
